## ğŸ†˜ Troubleshooting

### Links Not Working
- Check file paths are relative (e.g., `how-to-use.html` not `/how-to-use.html`)
- Verify filenames match exactly (case-sensitive on servers)
- Check that linked pages exist

### Styling Issues
- Verify `assets/css/style.css` is linked
- Check color values in inline styles
- Ensure `style` tags are closed properly

### Mobile Display Issues
- Check viewport meta tag
- Test grid layouts at different widths
- Verify media queries are working

### Search Not Working (FAQ)
- Check `faqSearch` input ID
- Verify `faq-item` class names
- Test with Firefox if Chrome has issues
